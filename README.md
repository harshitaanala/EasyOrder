# EasyOrder - HappySouls Cafe

EasyOrder is a simple, user-friendly menu-based ordering system built using Streamlit and Python. It allows customers to browse a menu, select items, place orders, generate receipt and provide feedback—all within an intuitive web interface.

# 🔍 Features
✅ User Authentication – Customers enter their name and phone number to personalize their order experience.

✅ Dynamic Menu – Categorized food & beverage items with easy selection options.

✅ Flexible Ordering – Users can choose item quantity, update selections, and modify orders before checkout.

✅ Order Summary & Receipt – Displays real-time bill calculations, including subtotal, tax (15%), and applicable discounts.
